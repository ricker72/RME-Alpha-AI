# Modernized RME Brush Icons — Tibia-oriented UI

This pack modernizes the supplied brush icons while preserving compatibility.

## Compatibility preserved
- Original filenames
- 32x32 and 16x16 dimensions
- RGBA transparency
- Existing folder layout

## Visual changes
- Dark pixel-art interface suitable for a modern map editor
- Cyan brush-selection language for clearer size recognition
- Medieval wooden doors and framed windows
- Shield-based zone markers
- Improved contrast at both icon sizes

## Installation
Back up the existing `brushes` folder, then replace it with the folder from this archive.
